# How to see evaluated result of an expression in command in Development tool?
Step 1:

Open development tool in web browser.

Step 2:

navigate to `Console` tab.

Step 3:

type `allow pasting` in `Console` tab (if the `pasting` is NOT enabled)

Step 4:

type your expression in `Console` tab and click `enter` key to execute it
